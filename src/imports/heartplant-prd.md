# 产品需求文档 (PRD): 心植 (HeartPlant) IoT 社交小程序

## 1. 产品概述
**定位：** 一款基于情感连接的物联网（IoT）植物协同认领平台。
**核心理念：** 通过“情感镜像”设计，将植物的生长状态与用户间的亲情、爱情、友情深度绑定。

---

## 2. 核心功能模块

### 2.1 情感驱动的 UI 系统 (Emotional UI Skins)
根据认领时选择的情感类型，全局切换视觉风格：
* **亲情模式 (Kinship):** 温暖橙色调，大字体，圆润组件，强调安全感与陪伴。
* **爱情模式 (Romance):** 粉粉紫调，微光动效，呼吸感布局，强调浪漫与共鸣。
* **友情模式 (Friendship):** 翠绿/亮黄调，涂鸦风格，非对称布局，强调活力与趣味。

### 2.2 首页 (Home)
* **顶部搜索:** 支持植物品种、情感关键词搜索。
* **Banner 运营位:** 轮播展示“本月明星植物”或“感人故事”。
* **认领列表:** * 卡片式设计。
    * 显示：植物实拍图、实时健康度、认领人头像（多人）、情感标签。
    * **动态提醒:** 若传感器报警告（如缺水），卡片边缘会有脉冲动效。

### 2.3 互动详情页 (IoT Interaction)
* **实时视频:** 调用 IoT 摄像头流，实时查看植物。
* **传感器面板:** * 温度 ($T$): 显示摄氏度。
    * 湿度 ($H$): 土壤水分百分比。
* **远程控制:** * **浇水/施肥:** 点击触发硬件水泵/施肥器。
    * **情感留言:** 文字同步到植物侧的电子墨水屏。
* **心跳共振:** 若共同认领人同时在线，界面触发特殊的视觉交互。

### 2.4 互动社区 (Moment)
* **成长日志:** 自动聚合传感器异常点（如开花时的温度）的照片。
* **私密时间轴:** 仅共同认领人可见的心情文字记录。

---

### 2.5 互动大厅
* **分享故事**  分享成长/情感故事，提供评论功能

## 3. 页面结构表 (Site Map)

| 页面 | 核心元素 | 交互重点 |
| :--- | :--- | :--- |
| **Tab 1: 首页** | 搜索框、Banner、认领卡片流 | 快速进入互动入口 |
| **Tab 2: 互动** | 视频流、控制按钮、传感器仪表盘 | 情感化 UI 切换、硬件响应 |
| **Tab 3: 个人** | 我的植物、设备绑定、勋章墙 | 资产管理与荣誉激励 |

## 4. UI 生成 Prompt 指令 (供 AI 参考)

> **Prompt A (亲情版):** > 设计一个微信小程序界面，主题为“亲情植物认领”。使用暖橙色 #FFBB00，字体易读，包含实时温度和湿度的卡片，设计风格温馨、简洁。
>
> **Prompt B (爱情版):** > 设计一个浪漫风格的 IoT 控制界面。背景使用半透明磨砂玻璃效果，主色调粉紫色，加入动态水滴形状的浇水按钮，强调精致感和仪式感。
>
> **Prompt C (友情版):** > 设计一个潮流活泼的植物社交页面。使用亮绿色，加入插画元素，布局灵活，传感器数据以进度条形式展现，适合年轻人审美。

---

## 5. 技术约束 (Technical Specs)
* **通讯协议:** MQTT / WebSocket (实现低延迟实时互动)。
* **硬件接口:** 集成 ESP32/树莓派，支持 DHT11 (温湿度) 及水泵继电器。
* **前端:** 微信小程序原生框架 / Uniapp。
